# BodzioBot


##Opis
BodzioBot jest botem dla mojego serwera klasowego, klasy 1TI2 gr.B
Posiada on podstawową implementacje najpotrzebniejszych funkcji do 
--Podziału obowiążków
--Zarządzania czasem
--Planowania
--Sybszego i sprawniejszego odrabiania lekcji.

##Podstawowe
Prefix /
Flagi / lub -
Dzielenie parametrów
